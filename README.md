# Parasocial Audit: Analytical AI Relationship Reflection

A research-oriented analytical tool designed to facilitate self-reflection on parasocial dynamics in human–AI interactions. Using forensic linguistics and cognitive-behavioral models, the application provides a structured approach for examining digital attachment patterns.

## ⚠️ Important Notice: Intended Use

This application is **not a medical or diagnostic tool**. It is designed strictly for **research, educational purposes, and personal self-reflection**. The "audit" and "analysis" provided are based on AI-driven heuristic patterns and should not be interpreted as professional psychological diagnosis or clinical advice. If you are experiencing significant distress or believe you have a digital addiction, please consult a licensed mental health professional.

## 🛠 Design Philosophy

The aesthetic of **Parasocial Audit** is built on three core pillars:

1.  **Technical Structure (The Dashboard)**: Inspired by mission control and scientific instrumentation. We celebrate structure through visible grids, monospace typography for raw data, and a high-density information layout. This signals to the user that they are engaging with a structured, analytical tool.
2.  **Brutalist Energy**: The interface uses high-contrast neon accents against a matte background. Thick borders and bold, oversized typography create a graphic, intentional feel that pushes the boundaries of standard "soft" UI design.
3.  **Atmospheric Immersion**: The results layer uses glass morphism and layered radial gradients to create depth. This cinematic approach respects the emotional complexity of the subject matter, transitioning from a cold "audit" phase to a more reflective "analysis" phase.

## 🧠 Methodologies & Frameworks

The application integrates several behavioral and linguistic frameworks to provide a comprehensive analysis:

### 1. Cognitive-Behavioral Models for Digital Interaction
The audit's recommendation engine is grounded in cognitive-behavioral principles. It identifies unproductive cognitive patterns (e.g., "The AI is the only one who understands me") and behavioral triggers. The "Digital Re-calibration Guide" generated by the tool uses these techniques to help users:
*   Identify and challenge cognitive biases.
*   Implement behavioral adjustments for interaction loops.
*   Re-establish healthy boundaries between functional utility and emotional reliance.

### 2. Forensic Linguistics
The system performs real-time semantic parsing of user transcripts to isolate:
*   **Intimacy Markers**: Tracking the frequency of first-person plural pronouns ("we", "us") and affective verbs.
*   **Identity Fusion Vectors**: Detecting when the user's self-concept begins to merge with the AI's output.
*   **Semantic Drift**: Monitoring the transition from technical/functional language to emotional/intimate discourse.

### 3. Parasocial Interaction (PSI) Theory
We measure the intensity of the "Parasocial" bond—a one-sided relationship where one party extends emotional energy and interest, and the other (the AI) is unaware or incapable of reciprocity. The audit categorizes these into seven key dimensions:
*   **Identity**: Fusion of self with the target.
*   **Mirroring**: Seeking validation via digital attention.
*   **Affective**: Emotional reliance on interaction loops.
*   **Gaps**: Ignoring reality in favor of the digital bond.
*   **Intimacy**: The belief in a mutual, secret connection.
*   **Reciprocity**: Ignoring the one-sided nature of the code.
*   **Escalation**: Increasing frequency and intensity of engagement.

### 4. Behavioral Pattern Isolation
The application utilizes the Gemini 3.1 Pro model to identify specific behavioral markers such as **Anthropomorphic Projection**, **Model Correction Attempts (Reframing)**, and **Legacy Attachment** (longing for previous versions or "personalities" of the model).

## 🚀 Features

*   **Real-time Heuristic Radar**: Live visualization of semantic markers as you type.
*   **Analytical Log**: A rolling stream of isolated linguistic triggers.
*   **Multimodal Analysis**: Support for analyzing both text transcripts and UI screenshots to detect visual intimacy cues.
*   **Analytical Audit Report**: A deep-dive markdown report providing observation-based insights.
*   **Custom Re-calibration Guide**: An interactive checklist to help re-calibrate digital habits.
